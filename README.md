# Kira AI Studio

**Локальная AI-студия для Windows с чатами, долговременной памятью, Ollama и интеграцией ComfyUI.**

Kira AI Studio объединяет локального AI-ассистента, создание промптов, генерацию изображений и видео, исследования, работу с документами и инструменты обучения в одном desktop-приложении.

---

## Основные возможности

- локальный AI-чат на базе Ollama;
- долговременная память и история разговоров;
- проекты и задачи;
- создание и улучшение промптов;
- генерация изображений через ComfyUI;
- генерация видео через ComfyUI workflows;
- работа с документами и RAG;
- интернет-исследования;
- Brain, Study Room и Prompt Lab;
- голосовой режим;
- постоянный визуальный профиль персонажа;
- подготовка LoRA-датасетов;
- переключение языка интерфейса;
- локальное хранение пользовательских данных.

## Локальная работа

Приложение предназначено для работы на компьютере пользователя:

- локальные модели запускаются через Ollama;
- генерация выполняется через локальный ComfyUI;
- чаты, память и пользовательские данные хранятся локально;
- облачная подписка для основных функций не требуется.

Для интернет-исследований необходимо подключение к сети.

## Системные требования

- Windows 10 или Windows 11;
- установленный Ollama;
- установленный и настроенный ComfyUI;
- видеокарта, совместимая с выбранными моделями и workflows;
- объём VRAM зависит от используемых моделей;
- свободное место для локальных моделей и результатов генерации.
